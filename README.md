# 🎭 Feedback Mystery

**Feedback Mystery** is a premium, secure, and completely anonymous feedback platform. It allows users to create unique mystery links, share them across social media, and receive unfiltered, honest feedback from their circle without any identity trail.

## ✨ Features

- 👤 **100% Anonymous Messaging**: Receive feedback without knowing who sent it.
- 🔗 **Magic Link Generation**: Create your own unique URL to share on Instagram, WhatsApp, Discord, etc.
- 🤖 **AI Question Suggestions**: Powered by AI to help you nudge users with engaging questions.
- 🧊 **Premium Glassmorphic UI**: Modern dark-mode experience with smooth animations and mesh gradients.
- 🛡️ **Privacy First**: End-to-end encrypted feel with zero tracking on sender identities.
- 📊 **Secret Dashboard**: Manage and read all your messages in a clean, organized panel.

## 🚀 Tech Stack

- **Frontend**: [Next.js 15](https://nextjs.org/), [React 19](https://react.dev/), [Tailwind CSS 4](https://tailwindcss.com/)
- **State & Animation**: [Framer Motion](https://www.framer.com/motion/)
- **Backend / Database**: [MongoDB](https://www.mongodb.com/) with [Mongoose](https://mongoosejs.com/)
- **Authentication**: [NextAuth.js](https://next-auth.js.org/)
- **Components**: [Shadcn UI](https://ui.shadcn.com/)
- **Validation**: [Zod](https://zod.dev/)
- **Email**: [Resend](https://resend.com/)

## 🛠️ Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Abuhanan-source/my-feedback-mystery.git
   cd my-feedback-mystery
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up Environment Variables**:
   Create a `.env` file in the root and add:
   ```env
   MONGO_URI=your_mongodb_uri
   NEXTAUTH_SECRET=your_auth_secret
   RESEND_API_KEY=your_resend_api_key
   OPENAI_API_KEY=your_openai_key
   ```

4. **Run the development server**:
   ```bash
   npm run dev
   ```

## 🌐 Deployment

This project is optimized for deployment on **Vercel**. 

1. Push your code to GitHub.
2. Link your repository to Vercel.
3. Add your Environment Variables in the Vercel Dashboard.
4. Deploy! 🚀
